Alerta de Conexión o Firewall de Red

El navegador no pudo conectar con el servidor de Google Gemini. Detalle del error:

Error de Conexión (Failed to fetch).

💡 ¿Cómo resolver este error en 1 minuto?
1. ¿Estás en la red de la oficina (Tecmilenio/Tec)? El firewall escolar o corporativo bloquea por seguridad el puerto de Google Cloud API. Prueba compartiendo datos desde tu celular a tu laptop por un instante para comprobar si funciona de inmediato.
2. ¿Usas Adblocker, uBlock o el navegador Brave? Estos sistemas bloquean de forma agresiva las peticiones directas de Inteligencia Artificial porque las confunden con anuncios. Desactiva los bloqueadores para esta URL.
3. API Key sin restricciones: Asegúrate de que la API Key que generaste en Google AI Studio no tenga restricciones de IP o de dominio.
